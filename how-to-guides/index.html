How To Perform WoE Tramsformation?

To perform WoE transformation of all the dataset, you need to utilize WoEDataPreparation function within the transform module. For the demostration puprposes we used the HELOC dataset from the kaggle.

We recommend to download data for further training.

from combat.transform import *

import pandas as pd
import numpy as np

data  = pd.read_excel('heloc_dataset.xlsx')
data['default'] = data['RiskPerformance'].apply(lambda x: 1 if x in ['Bad'] else 0 )
data = data.drop(columns = ['RiskPerformance'])

# split data into training and testing sets
y = data['default']
x = data.drop(columns = ['default'])

# specify special codes if necessary
special_codes = [-9, -8, -7]

Before we move to WoE Transformation procedure we need prepare meta information about the data utilized first. That means prepare an .xlsx file with 3 columns:

  1. Variable - name of variables as in the heloc data
  2. dtype - either numerical or categorical
  3. Expec - -1 if you expect negative coefficient for the Variable, +1 - if positive and 0 - if ambigious

Once the meta data is prepared, upload it. Specify Variable columns as index_col

df_sign = pd.read_excel("Sign Expec.xlsx",  index_col='Variable')

The transform module provides with a rich functionalilty for WoE transofrmation.
